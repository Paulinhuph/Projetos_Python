# 🐍 Desafios e Pequenos Projetos em Python

Coleção de projetos desenvolvidos em Python com foco em lógica de programação, automação e simulação de cenários reais. Este repositório representa minha evolução como desenvolvedor, explorando desde scripts básicos até sistemas com regras de negócio mais estruturadas.

## 🚀 Objetivo

Desenvolver habilidades em:

- Pensamento computacional e Resolução de problemas
- Estruturação de código e **Arquitetura de APIs**
- **Persistência de dados (SQL e NoSQL)**
- Automação e Manipulação de arquivos

## 📂 Projetos

### 🌐 Flask & APIs (StyleSync)

- **FlaskAPI-Study/**: Estudos de rotas, Blueprints e padrões de fábrica.
- **gerenciador_tarefas.py**: API para controle de produtividade.

### 🗄️ Banco de Dados (CRUD & Modelagem)

- **Gerenciador_Escolar-BD/**: Sistema completo integrando modelos e esquemas.
- **db_sql.py & db_mongo.py**: Práticas de integração com SQLite e MongoDB.
- **scripts de CRUD**: `create_db.py`, `read_db.py`, `update.py` e `write_db.py`.

### 📂 Manipulação de Arquivos

- **Processamento de Dados**: Scripts para lidar com arquivos `.csv`, `.json` e `.txt`.
- **Praticando.py**: Exercícios de leitura e escrita de fluxos de dados.

### 📊 Análise de Dados e Performance

## 🛠️ Tecnologias Utilizadas

- **Linguagem:** Python 3.x
- **Web/API:** Flask, Blueprints, Pydantic
- **Bancos de Dados:** SQLite (SQL), MongoDB (NoSQL)

- **Ferramentas:** Git, GitHub, venv (Ambientes Virtuais)
